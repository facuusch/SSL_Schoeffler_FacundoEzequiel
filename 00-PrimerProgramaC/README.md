El objetivo del **TP0** es tener un <ins> primer contacto con los sistemas de control de versiones GitHub y Git </ins>, el primero para familiarizarse con el uso de repositorios como sitio de nuestros proyectos y el segundo siendo la versión de control de nuestros archivos.

La idea es compilar y ejecutar un código que permita imprimir la frase “Hola mundo”.
